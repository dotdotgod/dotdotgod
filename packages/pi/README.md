# @dotdotgod/pi

Pi adapter for dotdotgod project memory workflows.

## Install

```bash
pi install npm:@dotdotgod/pi
```

For local development:

```bash
pi install /Users/dotdot/Workspace/dotdotgod/packages/pi
```

## Included

- `project-initializer` skill
- `plan-mode` extension
- `load-project` extension

## Commands

```text
/plan
/todos
/load
/dd:load
```

See the workspace root README for the cross-agent package map.
